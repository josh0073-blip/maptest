Drop any image files into this folder and the app will auto-seed them into
Background Library on launch.

Supported extensions:
- .png
- .jpg
- .jpeg
- .webp
- .gif
- .svg

Seed name behavior:
- Derived from filename (hyphens/underscores become spaces).
- Example: lfm-placement-map-sunday.png -> Lfm Placement Map Sunday

Notes:
- Keep filenames unique to avoid duplicate-looking entries.
- Recommended width is around 1400px to match the current map working area.
